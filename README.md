# Paws and Preferences

A mobile-first cat browsing app built with React, Tailwind CSS, and Motion. Swipe left or right to like or skip cats, and view a summary of liked cats. Images are sourced from [Cataas](https://cataas.com/).

## Features

- Swipe cards left or right to like/dislike cats
- Animated card transitions
- Tag overlays on images
- Summary of liked cats
- Refresh button to load a new batch
- Mobile-optimized layout

## Technologies

- React + Vite
- Tailwind CSS
- Motion
- Cataas API
